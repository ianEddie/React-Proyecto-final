## ⌨ Instalacion

Para utilizar la aplicación se deben ejecutar los siguientes comandos en consola.

```bash
  git clone https://github.com/ianEddie/React-proyectoFinal.git
```
```bash
  cd React-proyectoFinal
```
```bash
  npm i 
```
```bash
  npm run dev 
```
